# VoxTrans v0.1.0

VoxTrans 是一个桌面端字幕工作台，目标是把英文音视频快速变成可编辑、可导出的双语字幕。

## 核心能力

- 英文音视频转录（本地 ASR）
- 英译中字幕翻译（兼容 OpenAI 协议）
- 标点优化 + 识别矫正 + 断句优化（翻译后）
- 术语表筛选与翻译阶段注入
- 失败可重试、阶段可续跑、任务进度可追踪
- 字幕编辑器内查看与修改，支持导出多种 SRT 结果

## 典型流程

1. 导入本地文件或 YouTube 链接
2. 一键转录 / 转译
3. 在字幕编辑器中检查与微调
4. 导出 `src.srt` / `trans.srt` / `src_trans.srt` / `trans_src.srt`

## LLM 配置（兼容 OAI）

在设置中填写：

- `API Base URL`
- `API Key`
- `Model`

## 当前版本说明

- 当前主流程聚焦英文转录与英译中。
- 适合课程讲解、交易复盘、播客口播、YouTube 长视频等内容处理。
