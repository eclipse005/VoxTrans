# VoxTrans

VoxTrans 是一款桌面端音视频处理工具，聚焦于转录、翻译和字幕编辑。

它可以帮助你：

- 将音频/视频转录为带时间轴的字幕
- 通过 OpenAI 兼容接口进行字幕翻译
- 高效编辑并导出字幕文件
- 在同一流程中处理本地媒体与 YouTube 来源
