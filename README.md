# VoxTrans

VoxTrans 是一个桌面端字幕工作台，专注把音视频快速变成可编辑、可导出的双语字幕。

## 功能介绍

- 音视频一键转字幕
- 支持字幕翻译（英译中），可直接生成双语字幕
- 自动优化标点与断句，减少手动整理时间
- 支持导入 YouTube 链接或本地文件
- 支持任务进度查看、失败重试与断点续跑
- 内置字幕编辑器，可边看边改
- 支持多种字幕导出方式，方便发布与二次加工

## 使用步骤

1. 打开 VoxTrans，导入本地音视频或粘贴 YouTube 链接。
2. 选择需要的处理方式（转录或转录+翻译）。
3. 等待任务完成，在字幕编辑器中检查并微调内容。
4. 导出字幕文件并用于剪辑、发布或存档。
