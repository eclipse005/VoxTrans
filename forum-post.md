## 功能介绍

- **离线多语言转字幕** — 支持英语、中文（普通话/粤语）、日语、韩语、法语、德语、意大利语、西班牙语、葡萄牙语等，无需联网
- **AI 字幕翻译** — 需自行配置 LLM API（兼容 OpenAI 格式）
- **内置字幕编辑器** — 支持时间轴微调、合并/拆分、文本替换、批量操作
- **字幕压制** — 一键将字幕烧录到视频中
- **YouTube 下载** — 粘贴链接直接下载音视频并进入转录流程
- **任务队列** — 支持批量处理、进度查看、失败重试与断点续跑
- **一键在线更新** — 内置更新检测，有新版本时自动从 GitHub 下载安装，无需手动找链接

## 转录说明

- 转录基于 **Qwen3-ASR** 模型，**完全本地运行**，模型首次下载后永久离线可用
- 支持 **CPU / CUDA** 双后端，NVIDIA 显卡用户可在设置中切换 CUDA 加速
- 翻译功能依赖外部 LLM，需自行配置 API Key

## 使用步骤

1. 设置 → 模型管理中下载转录模型与对齐模型
2. 导入本地音视频，或粘贴 YouTube 链接
3. 设置源语言与目标语言，选择处理方式
4. 等待任务完成，在字幕编辑器中检查并微调
5. 处理完成后，输出文件位于安装目录 `output/` 文件夹
