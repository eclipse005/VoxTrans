## 功能介绍

- **离线多语言转字幕** — 支持中英粤日韩法德意西葡等语种，无需联网
- **AI 字幕翻译** — 兼容 OpenAI 格式 API，支持视觉辅助翻译（提取视频帧辅助理解语境）
- **内置字幕编辑器** — 时间轴微调、合并/拆分、文本替换、批量操作
- **字幕压制** — 任务完成自动烧录字幕到视频，内置多种样式预设
- **YouTube 下载** — 粘贴链接直接下载音视频进入转录流程
- **任务队列** — 批量处理、进度查看、失败重试、断点续跑
- **一键在线更新** — 内置更新检测，自动从 GitHub 下载安装

## 转录说明

- 三引擎：**Qwen3-ASR** / **Cohere Transcribe** / **MOSS-Transcribe-Diarize**，设置中切换
- **完全本地运行**，模型首次下载后永久离线可用
- 支持 **CPU / CUDA** 双后端，N 卡用户可切换 CUDA 加速
- 翻译功能依赖外部 LLM，需自行配置 API Key；支持 token 级流式预览

## 使用步骤

1. 设置 → 模型管理中下载转录模型与对齐模型
2. 导入本地音视频，或粘贴 YouTube 链接
3. 设置源语言与目标语言，选择处理方式
4. 等待任务完成，在字幕编辑器中检查并微调
5. 输出文件位于安装目录 `output/` 文件夹
