## v1.2.1

### 兼容性（重点）
- **CUDA 版大幅提升兼容性**：有 N 卡一般只需装好较新的显卡驱动即可加速，**不必再装 CUDA Toolkit**
- 驱动过旧（例如 nvidia-smi 里 CUDA Version 还是 12.3）请先升级驱动再试
- 常见游戏卡（含 20 系如 2080 Ti）在驱动足够新时可用 CUDA 版
- 安装包仍然不大；首次安装 CUDA 版会自动下载加速所需文件（需联网）

### 体验
- 模型管理去掉「一键下载当前 ASR + 对齐」按钮，改为每个模型单独下载
- 其他稳定性相关小修复

### 安装包
- `VoxTrans_1.2.1_cpu.exe` — CPU 版
- `VoxTrans_1.2.1_cuda.exe` — CUDA 版（需 NVIDIA 显卡 + 较新驱动）
