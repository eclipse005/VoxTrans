## v1.2.1

### CUDA（重要）
- **方案 B**：GPU 内核改为**预编译多架构 PTX**，运行时**不再依赖 NVRTC / CUDA Toolkit**
- 用户只需安装**较新的 NVIDIA 显卡驱动**（`nvidia-smi` 中 CUDA Version 建议 **≥ 12.6 / 12.8**；过旧驱动如 545 / CUDA 12.3 会报 PTX 工具链不支持）
- 安装包仍约 50MB 级；安装时自动下载 CUDA 12.8 计算库（cudart / cublas / cublasLt / curand），**不含** nvrtc
- 支持常见架构含 sm_61（Pascal）～ sm_90；RTX 20 系（如 2080 Ti / sm_75）在驱动足够新时可用

### 体验
- 模型中心去掉「下载当前 ASR 与对齐模型」一键按钮，改为各模型行单独下载
- 修复前端构建类型错误，保证安装包可复现编译

### 安装包
- `VoxTrans_1.2.1_cpu.exe` — CPU 版
- `VoxTrans_1.2.1_cuda.exe` — CUDA 版（需 NVIDIA 显卡 + 较新驱动）
